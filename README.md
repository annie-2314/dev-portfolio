# Devesh Roy — Insurance Advisor Portfolio

A fast, fully static, bilingual (English ⇄ తెలుగు) one-page portfolio for
**Devesh Roy · Roy Insurance Advisor**.

## Files (this is everything you need to keep)

| File          | Purpose                                             |
|---------------|-----------------------------------------------------|
| `index.html`  | The entire site — HTML, CSS and JS are all inline.  |
| `favicon.svg` | Browser tab icon (navy shield + people mark).       |
| `img.jpg`     | Hero image of Devesh Roy.                           |
| `README.md`   | This file.                                          |

There is **no build step** and **no dependencies** — just open `index.html`.

## Run locally

Double-click `index.html`, or serve it:

```bash
npx serve .
```

## Deploy on Render

Create a **Static Site** on [render.com](https://render.com):

1. **New → Static Site** and connect this GitHub repo.
2. **Build Command:** *(leave empty)*
3. **Publish Directory:** `.`  (a single dot — the repo root)
4. Click **Create Static Site**.

Render serves `index.html` at the root automatically. Every push to `main`
redeploys.

## Features

- English ⇄ Telugu language toggle (remembers choice via `localStorage`)
- Health / Motor / Life / Other insurance service sections
- Auto-sliding client testimonials
- Click-to-call, WhatsApp deep links, email, and a no-backend contact form
  that opens a prefilled WhatsApp chat
- Responsive, accessible, and reduced-motion friendly
